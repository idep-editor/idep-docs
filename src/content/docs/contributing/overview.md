Idep is Apache 2.0 and welcomes contributions.

## Where to start

- Browse [open issues](https://github.com/idep-editor/idep/issues)
- Read [TODO.md](https://github.com/idep-editor/idep/blob/main/TODO.md) for planned work
- Read [CONTRIBUTING.md](https://github.com/idep-editor/idep/blob/main/CONTRIBUTING.md) for guidelines

## Docs contributions

This docs site (`idep-editor/idep-docs`) is public and separate from the main repo.
Typo fixes, clarifications, and new pages are all welcome via pull request.

```bash
git clone https://github.com/idep-editor/idep-docs
cd idep-docs
pnpm install
pnpm dev
```
